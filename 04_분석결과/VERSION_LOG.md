## v2_sdot — S-DoT 실측 기반 (2026-04-01)

**파일**: `02_thermal_ppa_v2_sdot.py`
**결과**: `thermal_ppa_v2_sdot_summary.json`

### 방법
- S-DoT 57개 센서 실측 (2025.07.28~08.03, 7일 평균)
- UTCI 계산: tdb(실측) + rh(실측) + v=1.5m/s(가정) + tr=tdb+3.5×UV(추정)
- IDW(p=2) 보간 → 링크 15,608개 × 24시간 UTCI 매핑
- 교량 추가 패널티 × 0.7 적용

### 속도계수 (실측 기반, 시간대별 링크마다 다름)
| UTCI 범위 | 기본 계수 | 교량 계수 |
|---|---|---|
| < 26°C | 1.00 | 0.70 |
| 26–32°C | 0.90 | 0.63 |
| 32–38°C | 0.75 | 0.53 |
| 38–46°C | 0.50 | 0.35 |
| > 46°C | 0.10 | 0.05 |

### 결과
| 시간대 | 도달 노드 | PPA(%) | Classic 대비 |
|---|---|---|---|
| Classic | 2,529 | 45.7% | — |
| 07시 | 918 | 16.6% | **-63.7%** |
| 10시 | 916 | 16.6% | **-63.8%** |
| 13시 | 402 | 7.3% | **-84.1%** |

### v1 대비 변화 포인트
- **07시 PPA 급감**: v1은 07시를 쾌적(45.7%)으로 가정했지만,
  실제 폭염 시기 07시 UTCI ≈ 32°C(strong heat) → 이미 63.7% 감소
- **13시 더 심한 감소**: 84.1% (v1: 80.3%)
- **07시~10시 거의 동일**: 실측 UTCI 차이가 작아 PPA 차이도 미미

### 현재 한계 (추후 개선 예정)
- [ ] MRT = tdb + 3.5×UV → 1단계: ASOS 일사량 기반 개선
- [ ] 풍속 1.5m/s 가정 → ASOS 실측값으로 교체
- [ ] IDW: 센서 간 미기후 차이 미반영 → 2단계: SVF 기반 링크별 보정
