# 기상 데이터 수집 가이드

## 필요 데이터
- 기온(°C), 상대습도(%), 풍속(m/s), 일사량(MJ/m²)
- 시간별(1시간 단위)
- 대상 기간: 폭염특보 발령일 (여름철)
- 관측소: 서울 (지점코드 108) — 성동구 인근 ASOS 관측소

## 다운로드 방법

### 방법 1: API 자동 수집 (추천)
1. https://data.kma.go.kr 접속
2. 회원가입 → 마이페이지 → API 인증키 발급
3. `03_download_weather_api.py`에 인증키 입력 후 실행

### 방법 2: 수동 다운로드
1. https://data.kma.go.kr 접속
2. 데이터 > 기후통계분석 > 기상자료 > 지상(ASOS)
3. 지점: 서울(108)
4. 기간: 2022.06.01 ~ 2024.08.31
5. 요소: 기온, 상대습도, 풍속, 일사, 강수량
6. 다운로드 → `raw/seoul_asos_2022_2024.csv`로 저장

## 폭염 기준 (기상청)
- 폭염주의보: 일 최고기온 33°C 이상, 2일 이상 지속
- 폭염경보: 일 최고기온 35°C 이상, 2일 이상 지속
